## ๐ COMMON MISCONCEPTIONS & CORRECTIONS

| Misconception | What Students Think | Correction Strategy |
|---------------|-------------------|-------------------|
| **Counting errors** | Skipping numbers or counting twice | Use one-to-one correspondence; touch each object once |
| **Number recognition** | Confusing similar-looking numbers (6 and 9) | Provide clear visual models; practice daily |
| **Quantity mismatch** | Not understanding that 5 always means 5 objects | Use consistent manipulatives; show multiple representations |
| **Sequence gaps** | Not knowing what comes next | Practice the sequence daily; use number line |
| **Cardinality** | Repeating the last number instead of knowing the total | Emphasize "this many" when pointing to the last object |
